# Saantham

Saantham is a minimal digital experience focused on calm, clarity, and intentional interaction. It reimagines digital spaces by reducing noise and cognitive overload through clean design, thoughtful typography, and wide breathing spaces.

## Overview

A web experience designed to help users pause, breathe, and be. Saantham emphasizes empty space, gentle motion, and soothing interactions built specifically for caregivers—people who carry the responsibility of caring for others and often forget to pause for themselves.

## Features

- **Typography First:** Elegant, utilitarian typography driving emotional impact.
- **Editorial Layouts:** Mixed-grid designs, large image frameworks, and generous organic spacing.
- **Micro-interactions:** Calm, Apple-like easing (`0.8s cubic-bezier`) for hover states and button transitions.
- **Responsive Architecture:** Fully scalable design built with pure CSS Media Queries and Fluid Typography (`clamp()`).
- **Zero Dependencies:** Pure HTML and CSS providing a lightweight, ultra-fast viewing experience.

## Running the Project

Because this is a pure static site, there is no complicated build step required.

**To view the site locally:**
Simply open `index.html` in your favorite modern browser by double-clicking it, or use a tool like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VSCode for auto-reloading during development.
